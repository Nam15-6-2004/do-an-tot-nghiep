# NLShop - Website Kinh Doanh Quà Tặng Trực Tuyến
> Đồ án tốt nghiệp chuyên ngành Web Development

## 📝 Giới thiệu dự án
NLShop là một hệ thống website thương mại điện tử hoàn chỉnh, được nghiên cứu và phát triển độc lập từ khâu phân tích yêu cầu, thiết kế cơ sở dữ liệu (Database Schema) cho đến triển khai lập trình. Dự án nhằm tối ưu hóa trải nghiệm mua sắm quà tặng trực tuyến và quản lý đơn hàng hiệu quả cho doanh nghiệp nhỏ.

## 🛠 Công nghệ sử dụng
- **Frontend:** HTML5, CSS3, JavaScript, ReactJS
- **Backend:** NodeJS, Express framework
- **Database:** MySQL
- **Quản lý mã nguồn:** Git & GitHub

## ✨ Tính năng cốt lõi
- **Dành cho khách hàng:** Đăng ký/Đăng nhập, duyệt sản phẩm theo danh mục, tìm kiếm, giỏ hàng trực tuyến, đặt hàng và theo dõi trạng thái đơn hàng.
- **Dành cho quản trị viên (Admin):** Thống kê doanh thu, quản lý danh mục/sản phẩm (Thêm, sửa, xóa), quản lý và duyệt đơn hàng của khách.

## 📌 Điểm nổi bật về mặt kỹ thuật
- Thiết kế cơ sở dữ liệu chuẩn hóa, đảm bảo tính toàn vẹn dữ liệu và tối ưu hóa tốc độ truy vấn.
- Cấu trúc thư mục source code rõ ràng, áp dụng tư duy logic hệ thống cao.
